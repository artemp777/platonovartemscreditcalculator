# Калькулятор кредита

Веб-приложение для расчёта аннуитетных и дифференцированных платежей по кредиту.

## Функции
- Расчёт ежемесячного платежа, общей суммы и переплаты.
- Круговая диаграмма – соотношение основного долга и процентов.
- Таблица помесячных платежей (первые 6 и последний месяц).
- Сохранение до 5 последних расчётов в localStorage.
- Тёмная/светлая тема.

## Запуск
Просто откройте файл `index.html` в любом современном браузере (Chrome, Edge, Firefox, Safari). Для работы требуется интернет-соединение (подгружаются библиотеки через CDN).

## Структура
- `index.html` – главная страница
- `css/style.css` – кастомные стили (можно оставить пустым)
- `js/` – все скрипты:
  - `app.js` – контроллер
  - `calculator.js` – расчётные функции
  - `storage.js` – работа с localStorage
  - `chart.js` – построение графика
- `assets/screenshots/` – для скриншотов

## Технологии
- HTML5, CSS3 (Tailwind CSS)
- JavaScript (ES Modules)
- Chart.js
- Font Awesome
